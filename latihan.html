HTML (HyperText Markup Language)

HTML adalah bahasa standar untuk membuat halaman web. Pelajari cara menyusun konten, menambahkan gambar, tautan, dan banyak lagi.

1. Pengenalan HTML

Memahami apa itu HTML, sejarahnya, dan bagaimana ia bekerja dengan browser.

<!DOCTYPE html>
<html>...</html>

2. Struktur Dasar Dokumen

Mempelajari tag-tag esensial seperti `head`, `body`, `title`, dan meta tag.

<head>...</head>
<body>...</body>

3. Tag Teks & Formatting

Menggunakan heading, paragraf, bold, italic, dan elemen teks lainnya.

<h1>...</h1>
<p>...</p>

4. Link dan Gambar

Cara menyisipkan hyperlink dan gambar ke dalam halaman web Anda.

<a href="...">...</a>
<img src="...">

5. List (Daftar)

Membuat daftar berurutan dan tidak berurutan.

<ul><li>...</li></ul>
<ol><li>...</li></ol>

6. Tabel

Menyusun data dalam format tabel.

<table><tr><td>...</td></tr></table>

7. Form

Membuat formulir input pengguna.

<form><input type="text"></form>

8. Semantic HTML

Menggunakan tag HTML5 yang bermakna seperti `header`, `footer`, `nav`, `article`, `section`.

<header>...</header>
<footer>...</footer>

CSS (Cascading Style Sheets)

CSS digunakan untuk mengatur tampilan dan gaya halaman web. Pelajari cara membuat desain yang menarik dan responsif.

1. Pengenalan CSS

Apa itu CSS, mengapa penting, dan cara menghubungkannya dengan HTML.

<link rel="stylesheet" href="style.css">

2. Selector Dasar

Memilih elemen HTML menggunakan tag, class, dan ID.

.my-class { ... }
#my-id { ... }

3. Model Box

Memahami margin, border, padding, dan content.

margin: 10px;
padding: 20px;

4. Flexbox

Membuat layout responsif dengan Flexbox.

display: flex;
justify-content: center;

5. Grid

Membuat layout kompleks dengan CSS Grid.

display: grid;
grid-template-columns: 1fr 1fr;

String name = "Java";

2. Operator & Kondisi

Operator aritmatika, perbandingan, dan pernyataan `if-else`.

if (x > 5) { ... }

3. Loop

Perulangan `for`, `while`, dan `do-while`.

for (int i=0; i<5; i++) { ... }

4. Array

Menggunakan struktur data array.

int[] numbers = {10, 20, 30};

5. Fungsi (Methods)

Membuat dan menggunakan fungsi.

public void greet() { ... }

6. OOP Dasar

Konsep Class, Object, Encapsulation, Inheritance, Polymorphism.

class Car { ... }

SQL (Structured Query Language)

SQL adalah bahasa standar untuk mengelola dan memanipulasi database relasional.

1. Pengenalan SQL

Apa itu SQL, jenis-jenis perintah SQL (DDL, DML, DCL).

SELECT, INSERT, UPDATE, DELETE

2. Query Dasar (SELECT)

Mengambil data dari tabel dengan `SELECT` dan `FROM`.

SELECT column_name FROM table_name;

3. Filtering Data (WHERE)

Memfilter hasil query menggunakan klausa `WHERE`.

SELECT * FROM users WHERE age > 25;

4. Sorting Data (ORDER BY)

Mengurutkan hasil query secara ascending atau descending.

SELECT * FROM products ORDER BY harga DESC;

5. Agregasi (COUNT, SUM, AVG, MIN, MAX)

Melakukan perhitungan pada data.

SELECT COUNT(*) FROM orders;

6. JOINs

Menggabungkan data dari beberapa tabel.

SELECT * FROM orders JOIN customers ON ...;
